# CRYPT – Treasures Sheet v0.1

## Core Principles

- Skatt ger Värde (score) och Noise (börda).
- Skatt ger inga stat-bonusar.
- Noise påverkar:
  - Flykt först
  - Dodge därefter
  - Träffchans sist
- Noise eskalerar exponentiellt.
- Effekterna visas aldrig som exakta procentsatser.

---

# Noise Model (Intern)

0 skatter → Ingen effekt  
1 skatt → Låg påverkan  
2 skatter → Märkbar påverkan  
3 skatter → Betydande påverkan  
4 skatter → Hybris-zon  
5+ skatter → Nästan suicidalt, men möjligt

# Design Guardrails

- Skatt påverkar aldrig:
  - Max HP
  - Rå ATK
- Noise påverkar endast precision, dodge och flykt.
- Skatt kan inte tappas fritt.
- Skatt kan endast överges vid särskilda tillfällen (framtida system).
- Hybris ska kännas smygande, inte annonseras.
