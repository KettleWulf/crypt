# CRYPT – Run Structure v0.1

## Core Identity

Du är en girig gravplundrare.

Du går djupare för makt och rikedom.
Du kan alltid försöka vända.
Men beslut har konsekvenser.

---

# Run Length

Mål: 15–20 minuter  
Steg: ~20 djupsteg  
1 Väktare per run (bestäms vid start)

---

# Phase Structure

## Phase 1 – Lockelse (Steg 1–5)

- Låg risk
- Första skatt
- Första hint om väktare
- Spelaren känner kontroll

Syfte:
Bygga trygghet.

---

## Phase 2 – Tryck (Steg 6–12)

- Fler monster
- 1–2 sidospår
- Fler hintar (totalt 3–5 per run)
- Noise börjar märkas

Syfte:
Skapa girighet + osäkerhet.

---

## Phase 3 – Smygande Hybris (Steg 13–17)

- Högre risk
- Flykt mindre pålitlig
- Dodge börjar svikta
- Väktarens närvaro känns

Spelaren står inför val:

- Gå djupare
- Vända om

---

## Phase 4 – Slutval

Spelaren kan:

A) Fortsätta till Väktaren  
B) Vända och starta Retreat  

---

# Retreat System

Beslutet är definitivt.

När spelaren vänder:

- 3 Retreat-steg
- Inga sidospår
- Högre aggression
- Noise påverkar mer
- Väktaren kan inte spawnas

Om spelaren överlever 3 steg:
→ Feg vinst (score = skattvärde)

Om spelaren dör:
→ Allt förlorat

Narrativt kan väktaren antydas vid död.

---

# Guardian System

- Väktare bestäms vid run-start
- Hintas 3–5 gånger genom miljötext
- Ingen explicit bekräftelse

Om spelaren väljer konfrontation:

→ Bossfight (4–6 rundor)

Vid vinst:
→ Väktarens Sigill (extrem score)
→ Run slutar

Vid förlust:
→ Allt förlorat

---

# Win States

1. Dör i djupet  
2. Dör under retreat  
3. Överlever via retreat  
4. Besegrar väktaren  

---

# XP & Leveling

- 1 XP per steg
- +1 XP per besegrat monster
- ~5 levels per run
- Levels ger små förbättringar
- Items är viktigare än levels

---

# Design Guardrails

- Boss är aldrig obligatorisk
- Retreat är alltid riskabel
- Noise är smygande
- Spelaren ska känna:
  "Jag pressade för långt."
